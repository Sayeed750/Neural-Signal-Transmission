# 🧠 Neural Activity Visualization

## Overview
A 3D neural network simulation using PyOpenGL.  
It shows neurons, their connections, and moving signals between them.

---

## Features
- 3D neurons (nodes)
- Connections between neurons
- Moving signal pulses
- Glow effect using blending
- Depth effect using fog
- Rotating animation
- Camera movement

---

## Technologies
- Python  
- PyOpenGL  
- GLUT  

---

## Controls
- **W** → Forward  
- **S** → Backward  
- **A** → Left  
- **D** → Right  
- **ESC** → Exit
